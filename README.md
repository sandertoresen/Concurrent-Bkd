# Concurrent Bkd-tree Insert tests

This branch contains the results of the insert tests conducted with global structures.

## Vtune tests

- **VtuneBenchmark**: This folder includes the Intel Vtune projects used in the thesis benchmark.

## Scripts

- **InsertTestScript.py**: This script is used to run the insert tests on both the laptop environment.

## Graphs and Results

- **globInsertSmallBuff.py**: This file contains the Pyplot figure based on the inserter results.
- **globInsertBigBuff.py**: This file contains the Pyplot figure based on the inserter results.
- **resultInsert.txt**: This file contains the raw printout of the insert results generated by the `.sh` tests.
- **results_inserts.txt**: This file contains additional raw results generated by the `InsertTestScript.py`.

## Configuration
The tests were run with different configurations. The initial test were run with a THREAD_BUFFER_SIZE of 4096 and a GLOBAL_B_CHUNK_SIZE of 64. 64 trees were created.
For the larger test, a THREAD_BUFFER_SIZE of 16384 and a GLOBAL_B_CHUNK_SIZE of 16 were used. Similarly, this test also created 64 trees.
